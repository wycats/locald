# RFC: Dashboard as Workspace

## 1. Summary

Treat the dashboard as an active workspace, not a passive monitor.

## 2. Motivation

Users want to _do_ things (restart, stop, filter), not just watch.

## 3. Detailed Design

Add controls. Support keyboard shortcuts.

### Terminology

- **Workspace**: A place where work happens.

### User Experience (UX)

Interactive dashboard.

### Architecture

Frontend logic.

### Implementation Details

Svelte actions.
